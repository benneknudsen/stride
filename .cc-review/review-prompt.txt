Du er en grundig code reviewer. Gennemgå HELE dette repo (ikke kun git-diff). Brug dine subagenter til tre parallelle pass.

Løbende skriv alle fund til .cc-review/findings.md i repoet (kontinuerligt, ikke kun til sidst). Hvert fund skal være én blok:

## <titel>
**Beskrivelse:** <kort>
**Sted:** <fil:linje>
**Alvorlighed:** <low/med/high>
**Forslag:** <kort>

Pass 1 — VALIDERING:
- Kør npm run build, npm run check, npm run test, npm run lint
- Rapportér hvad der er grønt/rødt
- Tjek type-fejl med tsc --noEmit

Pass 2 — OPTIMERING:
- Unødvendige React re-renders
- Forkert brug af hooks
- Tailwind-dubletter
- Døde imports
- Overflødige dependencies
- Performance-flaskehalse

Pass 3 — BUGS & SIKKERHED:
- any-typer
- Memory leaks
- Usaneret input
- Unødigt tunge dependencies
- Sårbarheder (CSP, auth, crypto)
- Fejlhåndtering mangler

Write EVERY finding as you discover them to .cc-review/findings.md — do not wait until the end. This is critical because rate-limits may interrupt.
